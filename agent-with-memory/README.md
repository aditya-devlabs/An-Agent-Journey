# agent-with-memory

Tinkering with [mem0](https://mem0.ai) for storing user information, facts, experiences, preferences, and other personal context to give LLMs persistent memory across sessions.

### Setup

```bash
uv sync
```

Copy `.env` and add your API keys:

```
NVIDIA_API_KEY=
mem0-api-key=
```

### Usage

```bash
uv run main.py
```
